# Django APP & URLS

주어진 세개의 HTML 파일을 Django를 활용하여 사용자에게 제공하도록 하고, 링크를 통해 다른 HTML로 이동할 수 있도록 조정하도록 합니다.

```html
<!-- index.html -->
<h2>추가 페이지</h2>
<ul>
    <li>
        <a href="pokeball.html">몬스터볼</a>
    </li>
    <li>
        <a href="lorem.html">로렘입숨</a>
    </li>
</ul>
```

```html
    <a href="index.html">뒤로가기</a>
```
